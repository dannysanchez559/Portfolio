Recent Projects

Spoilist

My first iOS App! Spoilist lets a user add grocery items with user specified expiration dates or auto completed recommendations.
Spoilist will send a notification before items run out of time in order to reduce spoilage and save money!

Technologies implemented:

  • Local and remote repository with MacOS Terminal & Github
  • Design process, from wireframe, to Hi def mockup with Figma
  • Xcode Storyboard UI, Swift 4.0+, and MVC programming patterns
  • C.R.U.D. local data persistence using Realm with CocoaPod
  • Constraints and certain UI elements coded manually without Storyboard UI
